# Url Link Shortener App - Portfolio Project

#Portfolio #Tkinter #Python   
This is a simple URL shortener developed in Python using the Tkinter graphical library. This application allows you to take long URLs and transform them into shorter, more shareable versions.

### Features:
- URL Shortening: Convert long links into concise URLs.
- Copy to Clipboard: Copy the shortened URL directly to the clipboard with a single click.
- Intuitive Graphical Interface: User-friendly interface built with Tkinter.
